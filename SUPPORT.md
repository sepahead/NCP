# Support

NCP is pre-1.0 research software maintained by **Sepehr Mahmoudian** (**@sepahead**).

- **Bugs / feature requests** — open a [GitHub issue](https://github.com/sepahead/NCP/issues) using the templates.
- **Questions / ideas** — use [GitHub Discussions](https://github.com/sepahead/NCP/discussions).
- **Security** — see [`SECURITY.md`](SECURITY.md); please do not file public issues for vulnerabilities.

Good places to start: the spec (`NEURO_CYBERNETIC_PROTOCOL.md`), the design
[`RATIONALE.md`](RATIONALE.md), and the [`ROADMAP.md`](ROADMAP.md).
