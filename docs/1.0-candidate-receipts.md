  workspace tests, generation, installed packages, C++ demo,
  bounded pure-Python JSONL ingress, corpus/schema/proto/digest/baseline,
  security/profile validators, archives, dependency policy, and Buf. The lock used
  non-yanked `spin` 0.9.9/0.10.1.
  Because the source was still a dirty worktree, this is complete local preflight
  evidence but not an immutable release receipt.

  TypeScript binding exporters, 21 C ABI unit tests, 32 Zenoh unit tests, 5 gateway
