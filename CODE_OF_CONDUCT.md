# Code of Conduct

The NCP project adopts the **Contributor Covenant, version 2.1** as its Code of
Conduct. The full text is published at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>

**In short:** be respectful and considerate, assume good faith, welcome newcomers,
and give and receive feedback gracefully.

**Reporting:** to raise a concern, contact the maintainer (**@sepehrmn**) privately
via a GitHub security advisory. Reports are handled promptly and confidentially.

This Code of Conduct is adapted from the Contributor Covenant v2.1; see the linked
page above for the full community standards and enforcement guidelines.
